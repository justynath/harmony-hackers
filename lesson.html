Instructions for note flashcard game!

The aim of the game is to get as many treble clef notes right within a set time limit. The way the game works is that once you click the start game button, a flashcard shows at the top of the screen.
The user clicks a key and if they get it right, the key turns green and a correct message appears under the flaschard. If the user gets the note incorrect, the key will highlight in red and an incorrect message will appear under the flaschard.
When the game is over a pop-up will appear with a score. At any point during the game, you can press the stop button to end the game.

Hope you enjoy your musical journey!
